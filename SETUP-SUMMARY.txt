================================================================================
🐛 TROUBLESHOOTING QUICK TIPS
================================================================================

❌ MySQL connection failed?
   → Start XAMPP MySQL service
   → Check backend/.env configuration
   → Run: npm run reset

❌ Port already in use (5000 or 3000)?
   → taskkill /im node.exe /f
   → Or change PORT in backend/.env

❌ npm ERR! or module not found?
   → Run TROUBLESHOOT.bat → Option 3 (Clean install)
   → Or: cd backend && rmdir /s /q node_modules && npm install

❌ Login failed (wrong credentials)?
   → Run: npm run reset
   → Use: admin@toko.com / password123

❌ Frontend not connecting to backend?
   → Check backend is running (terminal should say "Server berjalan...")
   → Hard refresh browser: Ctrl+Shift+R
   → Clear cache: F12 → Application → Clear Storage

For more help: See GETTING-STARTED.md or run TROUBLESHOOT.bat
